Lab Four: Code Inspection and Class creation

Due Date: Feb 20, 2015

Objective

The objective of this lab is to

  • practice reading code,
  • correct some errors, and
  • complete the project.

 

Exercises

  1. Save and decompress the Netbeans project Lab4.zip
  2. Open it up in Netbeans.
  3. You should have four classes – Cart.java, Item.java, Register.java, Lab4.java.
  4. For now, ignore Register.java.
  5. You are to go through the code by reading and understanding or compiling it to find any syntax errors, run time errors or the logic errors.
  • There is some code in these classes that we have not covered – this is about understanding what you are looking at even if you don’t feel comfortable doing it yourself. This happens often where we have to understand what someone else has done.
  1. Finally, you are to complete Register.java. Look at the main, the code is commented out, and complete the code stub provided for your in Register.

    I have included a snapshot.jpg for you to see what your final output should resemble.

Hints

  • Things to look for are variable names, variable types, semicolons, parameter passing, return types, indexes, and particularly look for logic errors.
